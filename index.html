The graph above shows the most effective drug for each bacteria in the dataset. The visual encodings I used here are position along a common scale, a small scale, and the opacity and hue of each circle to highlight which drug is the most effective. I chose to use position along a common scale with bubbles because the exact value for each bubble isn’t as important as the underlying question of which bubble has the lowest position (relative to the x-axis) in each column, where each column represents a different bacteria. I used hue because using hue as a perceptual tool is very effective for categorizing nominal data (in this case the three different drug strains). In tandem with hue, I used the opacity of each circle to actually express if that particular circle had the lowest value for its column. The difference between two nearly opaque circles and one completely saturated circle in each column makes it very easy to pick out the saturated circle which is the circle that is intended to be highlighted. A small range of numbers on the y-axis was used for the same reason bubbles were used with position; that is because the exact value of the bubble isn’t as important as knowing if it is the lowest bubble or not. In having all the bubbles on one chart, besides knowing which drug was the most effective for each individual bacteria, you can also see how many bacteria each drug was the most effective on based off how much of each of the three colors you see which is something that would be much harder to get out of staring at tubular data.
